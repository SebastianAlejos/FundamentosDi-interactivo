Examen Parcial 1

Front End

El front end es el lado del sitio web que es visible para el cliente. Es la parte visual e interactiva del sitio web, y se utiliza HTML, CSS y JavaScript para su desarrollo.

Back End

El Back End es el responsable de guardar y organizar datos. Es el responsable de la comunicación entre el servidor y el sitio web.

End user

El End user es el cliente o usuario que va a utilizar el producto día con día. Es decir, si el sitio web es una tienda, el end user es el comprador de dicha tienda.

Client

El cliente es la compañía para la cual estás realizando el sitio. El cliente es el que decide cómo quiere que se vea su sitio web y como sea la interacción con el cliente.

En el ejemplo de la tienda, el cliente sería el dueño de la misma.

CDN Architechture

El Content Development Network es una solución al problema de la lentitud en muchos sitios web, ya que al acceder al sitio se necesitan realizar varios 'saltos' para contactar al servidor debido a la distancia física entre la computadora accesando al sitio, y el servidor.

Para solucionar este problema lo que hace el CDN es guardar versiones de tu sitio en servideros en todo el mundo para que la distancia de acceso sea menor y así el tiempo de carga sea menor para el End User

Historia

HTML

  • 1991- Se publica el primer documento formal con HTML con el nombre 'HTML Tags'.
  • 1995- IEFT(Internet Engineering Task Force) publica el éstandar HTML 2.0, primer éstandar oficial.
  • 1997- W3C(World Wide Web Consortium) publica HTML 3.2, la primera recomendación de W3C. Incorporan últimos avances en páginas web como applets de Java, entre otras.
  • 1998- Se publica HTML 4.0, la cual incorpora hojas de estilo CSS, inclusión de scripts, tablas complejas, etc.
  • 2004- Apple, Mozilla, y Opera crean la asociación WHATWG(Web Hypertext Application Technology Working Group), que se centra en el estándar HTML 5
  • 2007- W3C decide retomar la estandarización de HTML y se una al WHATWG.
  • 2014- Se publica HTML 5 el cual incluye nuevas etiquetas.

    • CSS

  • 1995- Hakon Wium Lie y Bert Bos se unen y crean CSS(Cascading Style Sheets).
  • 1995- W3C añade CSS al grupo de trabajo de HTML.
  • 1996- W3C publica la primera recomendación oficial, la "CSS nivel 1".
  • 1997- W3C separa el grupo de trabajo de CSS del de HTML.
  • 1998- Se publica "CSS nivel 2".
  • 2014- Se desarrolla CSS 3 el cual continúa en desarrollo.
